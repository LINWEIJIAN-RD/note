查看gpio的使用情况
cat /sys/kernel/debug/gpio


./pwm_test.sh 0 10000 5000 0
The first parameter represents the PWM channel
The second parameter represents the PWM cycle
The third parameter represents the PWM duty cycle
The fourth parameter represents the use of polarity, with 0 indicating the default polarity and 1 indicating polarity reversal
If the script is used directly, the output will be generated by default

设备列表
ls /sys/bus/platform/devices
驱动列表
ls /sys/bus/platform/drivers

/sys/class/pir	逻辑层面，一般不涉及硬件的控制，会一有一个软连接到/sys/devices/platform/硬件设备里面控制具体硬件
/sys/devices/platform/ 硬件设备，提供具体操作功能属性文件,通过驱动控制
/dev/pir0	用户态 I/O 接口，比如poll/select/epoll ，ioctl等

设备中断号：cat /proc/interrupts

马达驱动参数信息：cat /proc/jz/motor/motor_info
